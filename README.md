# Lead Management Workflow Automation

## Overview
This project automates lead collection, follow-up emails, and lead tracking using n8n, Google Forms, Google Sheets, and Gmail API.

---

## Features

- Lead collection via Google Forms
- Automatic storage in Google Sheets
- Automated follow-up email system
- Lead status tracking
- CRM dashboard metrics
- Workflow automation using n8n

---

## Tech Stack

- n8n
- Google Forms
- Google Sheets
- Gmail API

---

## Workflow Architecture

Google Form
↓
Google Sheets
↓
n8n Trigger
↓
Send Automated Email
↓
Update Lead Status
